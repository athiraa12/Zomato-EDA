# Zomato Data Analysis (EDA)

## Overview
This project performs Exploratory Data Analysis (EDA) on Zomato restaurant data to identify trends in ratings, cuisines, city-wise distribution, and online delivery availability across countries.

---

## Dataset
- **zomato.csv** – Restaurant-level data including location, cuisines, ratings, cost, and votes  
- **Country-Code.xlsx** – Country code mapping file  
- **Total records:** 9,551 restaurants

---

## Tools & Technologies
- Python  
- Pandas, NumPy  
- Matplotlib, Seaborn  
- Jupyter Notebook  

---

## Key Analysis
- Data cleaning and encoding handling  
- Merged country metadata using country codes  
- Rating distribution and customer engagement analysis  
- Country-wise and city-wise restaurant concentration  
- Online delivery availability by region  
- Popular cuisine frequency analysis  

---

## Key Insights
- India contributes approximately 94% of the listed restaurants  
- Most restaurant ratings fall between 3.0 and 3.9  
- Online delivery is primarily available in India and UAE  
- NCR cities (New Delhi, Gurgaon, Noida) dominate listings  
- North Indian and Chinese cuisines are the most popular
